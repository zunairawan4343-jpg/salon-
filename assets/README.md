Assets folder

Place images and favicon files here:

- favicon.ico            -> small site icon (16x16 / 32x32)
- logo.png / logo.svg    -> brand logo used in header
- hero.jpg               -> large hero image (optimized)

Tips:
- Use compressed JPEG/WEBP for photos.
- Add social preview image at `social-preview.jpg` (1200x630) and update OG tags in the HTML head.
- For production Tailwind builds, remove the CDN and build a minimized CSS with purge.
